PnL Research

Solana On-Chain Data Analysis Tool

Features

  • Fetch token price data from Birdeye API
  • Get transaction history from Solscan Pro API
  • Process raw data into OHLCV format
  • Calculate PnL (Profit and Loss) for wallets
  • Display interactive K-line charts

API Endpoints

GET /api/health - Health check
GET /api/kline/{token} - Get OHLCV K-line data
GET /api/trades/{wallet} - Get wallet trade history
POST /api/sync - Sync data from on-chain sources
